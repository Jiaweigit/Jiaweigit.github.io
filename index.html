远程仓库

远程仓库

远程仓储

远程仓库用来共享版本库，多人开发的时候使用

远程仓库其实是一个裸仓库

裸仓库：没有.git文件夹，直接将所有的版本库信息暴露在文件夹下
裸仓库没有工作目录， 在裸仓库中不能进行开发工作

  1. 创建远程仓库
    git init –bare

  2. 获取远程仓库的版本
    git clone 远程仓库地址 本地文件夹名称

  3. 把本地版本推送到远程仓库
    git push 远程仓库地址 本地分支:远程分支

如果本地分支和远程分支同名，写一个即可
git push origin master

  1. 从远程仓库获取更新后的版本
    git pull
    如果不写远程地址和分支名称，默认获取origin地址的master分支
    git pull 远程仓库地址 要pull的分支

远程仓库的详细操作

在文件修改之后，如何将修改的内容推送到远程仓库中

  1. 要使用git pull来获取一下远程仓库中最新的代码
  2. 再使用git push将自己修改的内容推送到远程仓库中

如何获取服务器中更新了的内容

远程地址管理

远程地址管理

我们可以在每一个本地仓库中，将远程地址保存下来。

保存的方式是键值对的形式， 也就是说保存的地址可以有一个别名，那么以后在使用这个地址的时候，直接使用别名即可。

如何保存一个远程地址

git remote add 别名 远程地址

显示所有已经保存的地址

git remote

origin的说明

如果当前版本库，是通过git clone从远程仓库获取下来的，那么这个本地库会自动默认添加一个origin远程地址，指向就是git clone 命令后面的那个远程地址

字符串方法

字符串方法

charAt() 在该字符串中根据下标（索引）来找对应的字符

   var str = "123456";
var result = str.charAt(2);
    console.log(result); //3
    console.log(str);    //123456

charCodeAt() 在该字符串中根据下标找对应字符的字符编码

var str = "123456"
var result = str.charCodeAt(2);
console.log(result);    //51 
console.log(str);    //123456

toUpperCase() 将小写转换成大写

var str ="Ab你cfdgCC好";
var result = str.toUpperCase();
console.log(result);    //AB你CFDGCC好
console.log(str);    //Ab你cfdgCC好

toLowerCase() 将大写字母转小写

var str ="Ab你cfdgCC好";
var result = str.toLowerCase();//将大写字母转小写
console.log(result);    //ab你cfdgcc好
console.log(str);    //Ab你cfdgCC好

slice() 提取字符串的某个部分，并以新的字符串返回被提取的部分

参数 :开始截取位置,结束截取位置(不包含结束) 不支持颠倒顺序 支持负数
var str = "Web前端开发工程师";    
var newStr = str.slice(4,-1);
console.log(newStr); //端开发工程
console.log(str);    //Web前端开发工程师

substring() 提取字符串中介于两个指定下标之间的字符

参数 :开始截取位置,结束截取位置(不包含结束)比要提取的子串的最后一个字符在stringObject中的位置多1 支持颠倒顺序 不支持负数(当成0处理)其内容是从 start 处到 stop-1 处的所有字符
var str = "Web前端开发工程师";    
var str2 = str.substring(4,0);
console.log(str2);    //Web前
console.log(str);    //Web前端开发工程师

indexOf() 返回某个指定的字符串值在字符串中首次出现的位置

参数： 第一个参数  查找的字符串  第二参数（第二参数不写 默认为0） 开始查找的位置
如果要检索的字符串值没有出现，则该方法返回 -1。
var str = "www.baidu.com/index.html";
var result = str.indexOf("i",15);
console.log( result );    //-1

lastIndexOf() 返回一个指定的字符串值最后出现的位置，在一个字符串中的指定位置从后向前搜索

var str = "www.baidu.com/index.html";
var result = str.lastIndexOf("i",15);
console.log(result);    //14

fromCharCode() 可接受一个指定的 Unicode 值，然后返回一个字符串

var result = String.fromCharCode(25771);
console.log(result);    //撫

concat() 方法用于连接两个或多个字符串。

var str1 = "hello"
var str2 = "world"
var result = str1.concat(str2);
console.log( result );    //helloworld

replace() 方法用于在字符串中用一些字符替换另一些字符，或替换一个与正则表达式匹配的子串。

var str = "hello";
var result = str.replace(/o/,"oworld");
console.log( result );    //helloworld

split() 方法用于把一个字符串分割成字符串数组

var str = "hello world";
var result = str.split(" ");
console.log( result );    //["hello", "world"]

valueOf() 方法可返回 String 对象的原始值

var str = "hello world";
var result = str.valueOf();
console.log( result );    //hello world

数组方法

数组方法

push()

作用：在数组的末尾 添加一个或者多个元素
参数：需要添加进去的元素（一个或者多个）
返回值：新数组的长度
注意：原数组会发生改变
var arr = [1,2,3,4,5,6]
var result = arr.push(7,8,9);
console.log(result);    //9
console.log(arr);    //[1, 2, 3, 4, 5, 6, 7, 8, 9]

pop()

作用: 从数组后面删除1项
参数：无参数
返回值：删除的那一项
注意：原数组会发生改变
var arr = [1,2,3,4,5,6]
var result = arr.pop();
console.log(result);    //6
console.log(arr);    //[1, 2, 3, 4, 5]

unshift()

作用: 从数组前面往里面添加一个或者多个元素
参数：添加的一个或者多个元素
返回值：改变后的数组的长度
注意：原数组会改变
var arr = [1,2,3,4,5,6]
var result = arr.unshift(-1,0);
console.log(result);    //8
console.log(arr);    //[-1, 0, 1, 2, 3, 4, 5, 6]

shift()

作用：从数组中前面删除一项
参数：无参数
返回值：删除的那一项
注意：原数组会发生改变
var arr = [1,2,3,4,5,6]
var result = arr.shift();
console.log(result);    //1
console.log(arr);    //[2, 3, 4, 5, 6]

reverse()

作用： 用来反转数组的
参数 无参数
返回值： 反转之后的数组
注意：原数组会发生改变
var arr = [1,2,3,4,5,6]
var result = arr.reverse();
console.log(result);    //[6, 5, 4, 3, 2, 1]
console.log(arr);    //[6, 5, 4, 3, 2, 1]

sort()

作用: 数组排序的  按照字符编码进行逐位比较的
参数: 情况1  无参数  情况2 参数是一个匿名函数
返回值：排序好后的数组
注意：原数组会发生改变
var arr = [3,5,1,20,9,75];
var result = arr.sort();
console.log(result);    //[1, 20, 3, 5, 75, 9]
console.log(arr);    //[1, 20, 3, 5, 75, 9]

var arr = [3,5,1,20,9,75];
var result = arr.sort(function(a,b){
      return a-b;
});
console.log(result);    //[1, 3, 5, 9, 20, 75]
console.log(arr);    //[1, 3, 5, 9, 20, 75]

var arr = [ 1,2,3,4,5];
//打乱数组 随机
arr.sort(function(){
      return Math.random()-0.5;
})
console.log(arr);

concat()

作用： 用于连接数组的
参数： 数组或者元素
返回值：连接后的新的数组
注意:原数组 不会发生改变 
var arr = [1,2,3,4,5,6]
var arr2 = [7,8]
var num = 9;
var result = arr.concat(arr2,num);
console.log(result);    //[1, 2, 3, 4, 5, 6, 7, 8, 9]
console.log(arr);    //[1, 2, 3, 4, 5, 6]

slice()

作用：用于截取数组 返回新的截取后的数组
参数:第一个： 开始截取位置     第二结束截取位置（不包含结束位置）
返回值：返回新的截取后的新的数组
注意：原数组不发生改变 
var arr = [1,2,3,4,5,6]
var result = arr.slice(0,-1);
console.log(result);    //[1, 2, 3, 4, 5]
console.log(arr);    //[1, 2, 3, 4, 5, 6]

splice()

作用： 是用来删除数组中的元素
参数： 第一个： 开始删除的位置  第二个参数  删除的个数   第三个参数起  之后是  要添加的一个或者多个元素,从 index 所指的下标处开始插入
返回值:删除的那些元素组成的数组
注意:原数组会发生改变
var arr = [1,2,3,4,5,6]
var result = arr.splice(0,0,"a","b","c");
console.log(result);    //[]
console.log(arr);    //["a", "b", "c", 1, 2, 3, 4, 5, 6]

join()

作用：将数组中每一项 用连接符 连接 转成字符串
参数：指定要使用的分隔符。如果省略该参数，则使用逗号作为分隔符
返回值：返回一个字符串
注意：原数组不发生改变 
var arr = [1,2,3,4,5,6]
var result = arr.join("-");
console.log(result);    //1-2-3-4-5-6
console.log(arr);    //[1, 2, 3, 4, 5, 6]

toString()

作用：把数组转换为字符串，并返回结果
参数：无参数
返回值：返回一个字符串
注意：原数组不发生改变
var arr = [1,2,3,4,5,6]
var result = arr.toString();
console.log(result);    //1,2,3,4,5,6
console.log(arr);    //[1, 2, 3, 4, 5, 6]

valueOf()

作用：返回 Array 对象的原始值
参数：无参数
返回值：返回 Array 对象的原始值
注意：
var arr = [1,2,3,4,5,6]
var result = arr.valueOf();
console.log(result); //[1, 2, 3, 4, 5, 6]
console.log(arr); //[1, 2, 3, 4, 5, 6]

正则表达式

正则表达式

声明和测试方法

1.构造函数法;
var reg1 = new RegExp(/abc/);
console.dir(reg1); // /abc/
console.log(reg1); // /abc/
console.log(typeof reg1); // object

2.直接量;字面量;
var reg2 = /abc/;
var reg3 = /abc/;
console.dir(reg2); // /abc/
console.log(reg2); // /abc/
console.log(typeof reg2); // object

简单数据类型比较的时候,比较的是数值;
复杂数据类型比较的时候,比较的是地址值;
console.log(reg1 === reg2); //false
console.log(reg3 === reg2); //false

正则表达式的使用

test(); 测试参数是否符合正则表达式的约束; 符合就是true; 不符合就是false;
var reg1 = new RegExp(/abc/);
console.log(reg1.test(“abc”)); //true
console.log(reg1.test(“def”)); //false

五大内部类

  1. 预定义类:正则表达式预先设计好的一些元字符
    (1) . 查找单个字符，除了换行和回车以外;
    console.log(/./.test(“ “)); //true
    console.log(/./.test(“nmv”)); //true
    console.log(/./.test(“123”)); //true
    console.log(/./.test(“#$%^”)); //true
    换行和回车不行
    console.log(/./.test(“\r”)); //false
    console.log(/./.test(“\n”)); //false
    console.log(/./.test(“\n\r”)); //false
    console.log(/./.test(“”)); //false
    console.log(“a\r\na”);
    (2) \d 数字字符; \D 非数字;
    console.log(/\d/.test(“1233456”)); //true
    console.log(/\d/.test(“abc #%$^”)); //false
    console.log(/\D/.test(“1233456”)); //false
    console.log(/\D/.test(“abc #%$^”)); //true
    (3) \s 空白字符; \S 非空白字符;
    console.log(/\s/.test(“ “)); //true
    console.log(/\S/.test(“ “)); //false
    (4) \w 单词字符; \W 非单词字符;
    console.log(/\w/.test(“aA0“)); //true
    console.log(/\w/.test(“!@#$%^”)); //false
    console.log(/\W/.test(“aA0    “)); //false
    console.log(/\W/.test(“!@#$%^”)); //true

  2. 简单类
    (1)直接书写; (必须是完整的，只多不能少,顺序必须对)
    console.log(/abc/.test(“abc”)); //true
    console.log(/abc/.test(“xxabcxx”)); //true
    console.log(/abc/.test(“acb”)); //false
    console.log(/abc/.test(“ab”)); //false
    console.log(/abc/.test(“abxxxxc”)); //false
    (2)写到[]里; (只要包含里面的任何一个就可以)
    console.log(/[abc]/.test(“axxxxxxx”)); //true
    console.log(/[abc]/.test(“vbxxxxxxx”)); //true
    console.log(/[abc]/.test(“vyyyucxxxxxxx”)); //true
    console.log(/[abc]/.test(“vyyyuxxxxxxx”)); //false

  3. 负向类: 把^写到[]里面的最前面就是负向类;
    被测试的参数只要有正则中没有规定的就是true;
    如果完整包含,或者只是约束的一部分;那么就是fales;
    console.log(/[^abc]/.test(“abccr”)); //true
    console.log(/[^abc]/.test(“xxx”)); //true
    console.log(/[^abc]/.test(“abx”)); //true
    console.log(/[^abc]/.test(“abc”)); //false
    console.log(/[^abc]/.test(“ab”)); //false

  4. 范围类: - ; 从哪到哪的意思;一般与[]连用;
    console.log(/[a-d]/.test(“cc”)); //true
    console.log(/[a-d]/.test(“ecc”)); //true
    console.log(/[a-d]/.test(“e”)); //false
    console.log(/[0-9]/.test(“e”)); //false
    console.log(/[A-F]/.test(“e”)); //false

  5. 组合类: 就是把范围类,一起用;
    console.log(/[a-dA-D0-5]/.test(“cC3”)); //true

正则边界: 1.以什么为开始; 2.以什么为结束; 3.必须是这样,不能多也不能少;

^: 1.[]里面的负向类;    2.  进入//里面直接书写的^;以什么...开始;
$: 以什么...为结束;
想输入元素组就要用\符;
  1. 以abc为开始;
    console.log(/^abc/.test(“a”)); //false
    console.log(/^abc/.test(“ab”)); //false
    console.log(/^abc/.test(“abc”)); //true
    console.log(/^abc/.test(“abcd”)); //true
    console.log(/^abc/.test(“abcde”)); //true
    console.log(/^abc/.test(“acbxxx”)); //false
  2. 以abc为结束;
    console.log(/abc$/.test(“a”)); //false
    console.log(/abc$/.test(“ab”)); //false
    console.log(/abc$/.test(“abc”)); //true
    console.log(/abc$/.test(“xxxxabc”)); //true
    console.log(/abc$/.test(“abcde”)); //false
    console.log(/abc$/.test(“acbxxx”)); //false
    3.必须是这样,不能多也不能少;
    console.log(/^abc$/.test(“ab”)); //false
    console.log(/^abc$/.test(“abc”)); //true
    console.log(/^abc$/.test(“abcd”)); //false
    console.log(/^abc$/.test(“abcabc”)); //false

正则表达式-量词

量词四种:
1.*     重复零次或更多    (>=0)

2.+ 重复一次或更多次 (>=1)
3.? 重复零次或一次 (0||1) 要么有 要么没有
4.{} 重复多少次的意思

    • 重复零次或更多 (>=0)
      a : 代表a不出现也可以,出现也可以, 多少次都行;
      console.log(/^a      $/.test(“”)); //true
      console.log(/^a$/.test(“a”)); //true
      console.log(/^a      $/.test(“aa”)); //true
      console.log(/^a$/.test(“aaa”)); //true
      console.log(/^a      $/.test(“aaaa”)); //true
      console.log(/^a*$/.test(“aaaab”)); //false

    • 重复一次或更多次 (>=1)
      a* : 代表a出现至少一次;
      console.log(/^a+$/.test(“”)); //false
      console.log(/^a+$/.test(“a”)); //true
      console.log(/^a+$/.test(“aa”)); //true
      console.log(/^a+$/.test(“aaa”)); //true
      console.log(/^a+$/.test(“aaaa”)); //true
      console.log(/^a+$/.test(“aaaab”)); //false

      1. ? 重复零次或一次 (0||1) 要么有 要么没有
        a* : 代表a出现零次或一次都行;
        console.log(/^a?$/.test(“”)); //true
        console.log(/^a?$/.test(“a”)); //true
        console.log(/^a?$/.test(“aa”)); //false
        console.log(/^a?$/.test(“aaa”)); //false
        console.log(/^a?$/.test(“aaaa”)); //false
        console.log(/^a?$/.test(“aaaab”)); //false

4.{} 重复多少次的意思
(1). {4} 重复4次; (2). {4,} 重复4次以上; (3). {4,8} 重复4-8次;
{n} n次 （x=n）
{n,} 重复n次或更多 (x>=n)
{n,m} 重复出现的次数比n多但比m少 (n<=x<=m)

console.log(/^a{3}$/.test(“aa”)); //false
console.log(/^a{3}$/.test(“aaa”)); //true
console.log(/^a{3}$/.test(“aaaa”)); //false
console.log(/^a{3}$/.test(“aaaab”)); //false

console.log(/^a{3,}$/.test(“aa”)); //false
console.log(/^a{3,}$/.test(“aaa”)); //true
console.log(/^a{3,}$/.test(“aaaa”)); //true
console.log(/^a{3,}$/.test(“aaaab”)); //false

console.log(/^a{3,5}$/.test(“aa”)); //false
console.log(/^a{3,5}$/.test(“aaa”)); //true
console.log(/^a{3,5}$/.test(“aaaa”)); //true
console.log(/^a{3,5}$/.test(“aaaaa”)); //true
console.log(/^a{3,5}$/.test(“aaaaaa”)); //false
console.log(/^a{3,5}$/.test(“aaaaaaa”)); //false
console.log(/^a{3,5}$/.test(“aaaaaaab”)); //false

正则表达式-优先级

1.（） 2.| 3.^$ 4.量词
| 是或的意思;在正则中没有||; 满足一个添加就行;

正则表达式-三个函数

1.replace(); 替换字符串;
var str = “aaaBBBAAACCCaaa”;
console.log(str.replace(“aaa”,”ooo”)); //oooBBBAAACCCaaa
console.log(str.replace(/aaa/g,”ooo”)); //oooBBBAAACCCooo
console.log(str.replace(/aaa/gi,”ooo”)); //oooBBBoooCCCooo
i: 忽略大小写; g:全局替换;

2.trim(); 去除字符串两边的空格; ie678不支持;
var str = “ aaa bbb ccc “;
console.log(str); // aaa bbb ccc
console.log(str.trim()); //aaa bbb ccc

3.search(正则); 给字符查索引值;
var str = “abcdefg”;
console.log(str.search(/cd/)); //2

正则表达式-trim()封装

function trim(str){
return str.replace(/(^\s+) | (\s+$) /g,””);
}
var str = “ aaa bbb ccc “;
console.log(str) // aaa bbb ccc
console.log(trim(str)) //aaa bbb ccc

我的第一篇博客-hexo搭建个人博客

  1. 安装hexo
    npm install hexo-cli -g

  2. 创建一个博客文件夹
    在文件夹中执行 hexo init

  3. 在文件夹中直接执行 hexo s 在本地测试网页效果

新建文章

hexo new “文章名称”

这句命令执行完毕之后，会在source/_posts/ 中生成以文章名为文件名的markdown文件

编辑文章

找到对应的markdown文件，编辑内容即可

生成html

hexo g

执行这条命令之后，会在public目录下生成所有的静态页面，只要将该目录下所有的文件进行发布，发布到github上，就可以通过github的静态页面托管服务来访问页面了

如何使用hexo d 来发布网站呢

  1. 安装 hexo-deployer-git
    npm install hexo-deployer-git –save

  2. 配置网站根目录下面的_config.yml
    配置最下面的deloy节点
    deploy:

    - type: git
  repo: github静态页面托管项目的地址

2.1 在public目录下执行 git init 并进行一次提交

  1. 按照正常流程创建博客， 编辑博客

  2. 执行hexo g

  3. 在 public 目录下去执行 hexo d
